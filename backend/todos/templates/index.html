Hello, This my first Django -Project

This is a basic todo app built using Django web framework.

This project has functionalities like Create, Read, Update and Delete (CRUD) and user authentication and authorization, etc!

Built using Bootstrap version 4.6.2 components (Navbar, Jumbotron, Inline-forms, List-groups, etc.) To get started first login register to your account by clicking here or register a new account by clicking here. Or just use the navbar! Once logged in, you can use the website or the REST API to list, create, update or delete a todo.
